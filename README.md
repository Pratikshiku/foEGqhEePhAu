### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户(员工)、部门经理、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 公告资讯
5. 任务
6. 资料
7. 个人中心
8. 部门
9. 任务分配
10. 员工请假
11. 基本信息
12. 薪资


**部门经理功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 用户管理
6. 基本信息管理
7. 部门管理
8. 员工请假管理
9. 任务管理
10. 任务分配管理
11. 公告资讯管理
12. 薪资管理
13. 资料管理
14. 轮播图信息



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 部门经理管理
5. 用户管理
6. 基本信息管理
7. 部门管理
8. 员工请假管理
9. 任务管理
10. 任务分配管理
11. 公告资讯管理
12. 薪资管理
13. 资料管理
14. 基础数据管理（公告类型管理、请假类型管理、任务类型管理、任务类型管理、资料类型管理）
15. 轮播图管理
